DreamCollisionWorld

What

This is just a C wrapper for BulletCollsion. Dream'CollisionWorld' is a fedora tip to btCollisionWorld, a key object from the bullet physics simulation library.

Why

  • I need reliable 3D collision detection for this engine.
  • My project is written in C.
